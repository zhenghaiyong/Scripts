程序：交互播放网络电视脚本v0.1
作者：dogking （踏雪无痕）
日期：2007年4月2日
简介：通过读取你输入的数字选择相应的电视台
更新：2007年4月2日晚加入支持参数模式

功能介绍：
一个交互脚本，调用Mplayer播放教育网在线直播。

系统要求：
Mplayer，mencoder ,及相关解码包。

使用方法：
在~/.bashrc添加
alias tv="sh  ~/xxx/tv.sh"
xxx是你存放脚本的具体路径
然后运行 $:source ~/.bashrc
以后直接输入 $:tv 即可实现节目收看，录制！

命令参数：
$:tv -h  ;查看帮助
$:tv -a ;查看作者信息
$:tv  ;普通模式
$:tv x  ;快速模式，x为频道号
$:tv -l  ;频道列表
$:tv -r x;录制节目x
$:tv -k;终止录制

注意事项：
不要在播放的时候使用快进等功能（鼠标滚轮，方向键等），那会造成Mplayer退出。因为是在线直播，所以不能跳进。
